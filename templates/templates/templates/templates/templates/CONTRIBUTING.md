# Contributing

Thank you for your interest in improving this project.

This repository is focused on reusable AI workflow templates for creators.

## How to contribute

You can contribute by:

- Improving existing templates
- Adding new workflow templates
- Fixing unclear wording
- Adding examples
- Suggesting better structure

## Template guidelines

A good template should be:

- Clear
- Reusable
- Practical
- Easy to adapt
- Helpful for non-technical creators

## Suggested contribution format

When adding a new template, include:

1. Purpose
2. Workflow or structure
3. Example
4. Notes or limitations

## Project goal

The goal is to help creators use AI tools with more structure, consistency, and responsibility.
