# Short-Form Content Repurpose Workflow

A workflow for turning one core idea into multiple short-form posts.

## Purpose

This template helps creators repurpose one idea across YouTube Shorts, TikTok, Instagram Reels, and carousel posts.

## Workflow

1. Start with one core message.
2. Turn the message into a short hook.
3. Create a 30-60 second vertical video concept.
4. Write a caption for each platform.
5. Create a thumbnail or cover frame.
6. Add a simple call to action.
7. Track what performed best.

## Platform Adaptation

### YouTube Shorts

Focus on clarity, searchable wording, and strong retention.

### TikTok

Focus on emotional hook, direct language, and fast pattern interruption.

### Instagram Reels

Focus on visual identity, clean captioning, and shareable message.

### Instagram Carousel

Focus on one idea per slide with a clear emotional arc.

## Example

Core idea:

> Creators do not need more random prompts. They need repeatable workflows.

Short hook:

> Stop collecting prompts. Start building systems.

Video angle:

> Show how one idea becomes a video, carousel, and caption using the same workflow.
