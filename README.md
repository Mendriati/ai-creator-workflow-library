# AI Creator Workflow Library

Open-source workflow templates for AI-assisted music, video, prompt engineering, and creator operations.

## Purpose

This repository helps creators use AI tools in a more structured, repeatable, and responsible way.

It includes practical workflows, prompt templates, and production systems for:

- AI-assisted music creation
- Short-form video production
- YouTube, TikTok, and Instagram content workflows
- Prompt engineering
- Google Flow-style video prompting
- Creator operations and repeatable publishing systems
